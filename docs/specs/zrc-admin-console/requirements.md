# zrc-admin-console Requirements

## Purpose
Self-host admin UI for:
- viewing pairings and revoking access
- viewing audit logs
- managing dirnode and relay config
- managing update channels (optional)

## Must-have
- Authentication (local admin user, optional SSO later)
- RBAC basic roles (admin, operator, viewer)
- Safe defaults, minimal exposed info

## Non-goals
- Cloud multi-tenant billing (optional later)
