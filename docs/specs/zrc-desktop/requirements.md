# zrc-desktop Requirements (GUI viewer/controller)

## Purpose
User-friendly desktop app:
- show device list, online status, pairings
- start remote session
- display remote screen
- send input and manage clipboard/files

## Must-have (Phase 1)
- Connect to paired devices
- Show session screen (software rendering ok)
- Mouse move/click
- Basic keyboard
- Session stop

Phase 2+
- Multi-monitor UI
- Clipboard
- File transfer
- Chat
- Connection health + bitrate controls
